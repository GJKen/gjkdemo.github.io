# 使用方法
例如, 你可能有一个名为 project 的 GitHub 仓库, 里面有一个 static 文件夹, 存储了一些文件.

`https://cdn.jsdelivr.net/gh/[GitHub用户名]/[仓库名]@[标签或分支]/[文件路径]`
https://cdn.jsdelivr.net/gh/gjken/gjken.github.io/static/primer.min.css

| 字段 | 对应
|-|-|
[GitHub用户名] | 对应你的 GitHub 用户名
[仓库名] | 对应你的 GitHub 仓库名称
[标签或分支] | GitHub 标签或分支, 例如 `https://cdn.jsdelivr.net/gh/jquery/jquery@3.2.1/dist/jquery.min.js`<br>都不写默认访问最新版本.</br>
[文件路径] | 对应你的 GitHub 仓库文件路径
